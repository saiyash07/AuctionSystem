## 1. Introduction
The Auction Platform is a console-based application developed using C++ to demonstrate
the practical implementation of Data Structures and Algorithms (DSA). The system simulates
a real-world auction environment where users can manage auction items, register bidders,
place bids, search and sort data, and determine the winning bidder. The project combines
multiple DSA concepts into a single application to improve efficiency and data management.

## 2. Problem Statement
Traditional auction systems can become difficult to manage when the number of items and
bidders increases. Searching for items, processing bids, organizing auction data, and
determining winners manually can be time-consuming and inefficient. Therefore, an
automated solution is required that can efficiently manage auction operations using
appropriate data structures and algorithms.

## 3. Objectives
● To develop a simple auction management system using C++.
● To demonstrate the practical use of Data Structures and Algorithms.
● To implement searching and sorting techniques for efficient data handling.
● To manage bids using Stack and Queue data structures.
● To organize auction categories using Tree structures.
● To identify the highest bidder using a Priority Queue.
● To improve efficiency in data storage, retrieval, and processing.

- Working of the System
- The user enters auction item details.
- Bidder information is registered and stored.
- Bidders place bids on auction items.
- Queue is used to process bids in the order they are received.
- Stack allows undoing the most recent bid if required.
- Search operations help users find auction items quickly.
- Sorting algorithms organize auction data efficiently.
- BST manages item categories.
- BFS and DFS perform graph traversal operations.
- Priority Queue identifies the highest bidder and determines the winner.
